show()method 
public static function show(target:IFlexDisplayObject, parent:DisplayObject, modal:Boolean = true, popupX:Number, popupY:Number):void

弹出指定对话框，并加以弹出效果。

Parameters

target:IFlexDisplayObject — 弹出对象，必须是一个实现了 IFlexDisplayObject 接口的 DisplayObject 类型对象。
 
parent:DisplayObject — 弹出时的父组件。
 
modal:Boolean (default = true) — 是否以模态窗口的形式弹出，默认为 true。
 
popupX:Number (default = NaN) — 弹出窗左上角在屏幕中的x坐标，默认为NaN，即居中弹出。
 
popupY:Number (default = NaN) — 弹出窗左上角在屏幕中的y坐标，默认为NaN，即居中弹出。
