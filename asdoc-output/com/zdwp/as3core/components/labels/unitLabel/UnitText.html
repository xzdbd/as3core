自定义单位文本组件。

该组件定义了一个包含一个上标的单位文本组件，形如 km³/s，分为三部分组装，即：km + ³ + /s。

MXML SyntaxShow MXML Syntax
Hide MXML Syntax

下面代码演示如何使用 UnitText 类：

首先，引用命名空间 xmlns:labels="http://www.xzdbd.net/2013/flex/as3core"，调用如下：

	<labels:UnitText 
	        Properties
	        leftText="km"
	        superscript="3"
	        rightText="/s"
	 
	        Styles 
	        textBold="false" 
	        useBracket="false"
	        color="0x000000" />
